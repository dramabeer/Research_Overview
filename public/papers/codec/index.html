CODEC: Detecting Linear Correlations in Dense Clusters using coMAD-based PCA

Published on 2019/08/26

in LWDA

Maximilian Archimedes Xaver Hünemörder, Anna Beer, Daniyal Kazempour, Thomas Seidl

Abstract

The coMAD (co-median absolute deviation) is a measure for the joint median of two random variables. Previous experiments have shown that a coMAD-based PCA is more robust towards noise and outliers, yielding eigenvectors which represent linear correlation better than its covariance-based competitors. In this preliminary work we introduce CODEC - COrrelations in DEnse Clusters - a method for detecting linear correlations in dense clusters utilizing a coMAD-based PCA. The idea of CODEC is intriguingly simple: first a density-based clustering is performed using the well established clustering method DBSCAN. Then on each of the clusters PCA is performed. Instead of using the covariance matrix we use the coMAD matrix as a basis for performing PCA.
